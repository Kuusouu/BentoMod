# BentoMod Changelog

All our tasty little updates will be packed into this adorable bento box!

## [0.1.0] - 2026-07-23

### Freshly Prepared
- Welcome to BentoMod's first intentional pre-1.0 release!
- Back up your mod collection to a zipped bento box and keep track of when it was last packed.
- Automatically fetch the latest character and skin data in the background.
- Automatically install the UTOC Signature Bypass when BentoMod detects your game.
- Quickly filter mods by status and customize BentoMod with full HEX color support.

### Recipe Tweaks
- Established BentoMod as an independently maintained, Windows-only fork with its own installer, updater, and release kitchen.
- Improved keyboard navigation and made buttons, forms, and other controls easier to use.
- Polished the interface so menus, lists, and settings behave more consistently.

### Kitchen Fixes
- Fixed the UTOC Signature Bypass files included with BentoMod.
- Fixed several interface issues that could cause lists and controls to behave unexpectedly.
- Restored the intended checkbox layout in Settings and left-aligned the log drawer status.
- Kept the installer output visible instead of clearing the terminal window.

### Kitchen Cleanup
- Tidied up the interface code to make BentoMod easier to maintain and improve.
- Cleared out inherited features and files that no longer belong in BentoMod's pantry.
