# Installing Mods

A step-by-step guide to installing mods with Repak X.

## Method 1: Drag & Drop

1. Open Repak X
2. Drag your mod file(s) or folder(s) onto the app window
3. Configure each mod in the install panel (name, tags, format, etc.)
4. Click **Install**

## Method 2: File Picker

1. Open Repak X
2. Click the **split-button file picker** in the toolbar
3. Choose **File** or **Folder** mode
4. Select your mod assets
5. Configure and install

## Method 3: Browser Extension

1. Install the Repak X browser extension
2. Browse mods on NexusMods
3. Click the Repak X install button on any mod page
4. The mod will be sent directly to your Repak X app

## Install Panel Options

For each mod, you can configure:

- **Name** — custom display name
- **Tags** — searchable tags for organization
- **Destination folder** — where the mod gets placed
- **Compression** — enable Oodle compression
- **Format** — Legacy PAK or IO Store
- **Repak toggle** — enable/disable repacking
