# Browser Extension

Install mods directly from NexusMods with the Repak X browser extension.

:::info Coming Soon
The Repak X browser extension is currently in development and will be available soon for Chrome and Firefox.
:::

## Setup

1. Download the Repak X browser extension (Chrome or Firefox)
2. Install it in your browser
3. Make sure Repak X is running on your computer

## Usage

1. Browse mods on [NexusMods](https://www.nexusmods.com/marvelrivals)
2. Click the **Repak X** install button on any mod page
3. The mod will be sent to your running Repak X instance
4. Configure and install as usual from the install panel
