BentoMod is a **mod installer and manager** for Marvel Rivals, made with love and a lot of care. Think of it like a real bento box. Everything has its place, it looks adorable, and it's packed with good stuff. Install paks and uassets, manage your collection, handle mod conflicts, and keep your game looking exactly the way you want.

## 📦 Getting BentoMod

### Option 1 — The quick way (Releases)

Grab the latest portable build straight from [**Releases**](https://github.com/Kuusouu/BentoMod/releases/latest). Just download, unzip, and run. No fuss.

### Option 2 — The proper way (Installer)

Want BentoMod to live on your system like a real app with a proper entry in Settings, a Start Menu shortcut, and a clean uninstaller? We've got you covered.

Open **any** terminal — Command Prompt, PowerShell, or Windows Terminal and run:

```powershell
powershell -ExecutionPolicy Bypass -c "irm https://raw.githubusercontent.com/Kuusouu/BentoMod/refs/heads/master/scripts/Install-BentoMod.ps1 | iex"
```

That's it! The installer will take care of everything for you (with a cute easter egg). And when you're ever ready to move on, BentoMod ships with a built-in uninstall script that cleans up every last crumb from your system.
